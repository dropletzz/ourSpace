Doom asset slots
================

Put exported PNG files in this folder. The game already has canvas fallbacks, so missing files will not break it.

Spritesheets
------------

- enemy.png: 4 horizontal frames, each 64x128.
- weapons.png: 9 horizontal frames, each 160x90.
  - frames 0-2: pistol idle/fire/fire2
  - frames 3-5: shotgun idle/fire/fire2
  - frames 6-8: rifle idle/fire/fire2
- pickups.png: 3 horizontal frames, each 48x48.
  - frame 0: pistol
  - frame 1: shotgun
  - frame 2: rifle

Wall and flat textures
----------------------

Put 64x64 PNG textures in textures:

- textures/wall_1.png
- textures/wall_2.png
- textures/wall_3.png
- textures/wall_4.png
- textures/floor.png
- textures/ceiling.png

If you extract Doom WAD assets, rename the exported PNGs to these names.
